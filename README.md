# Tokopedia Search Filter
A simple Search Filter using Tokopedia API

## Usage

1. Pull Project
2. Pod install

## Library used

1. Alamofire
2. HexColors
3. SwiftyJSON
4. SwiftMessages
5. SDWebImage
6. SwiftRangeSlider
7. IQKeyboardManagerSwift

## Minimum Requirements

iOS 9.0 or higher
Swift 4
XCode 9
